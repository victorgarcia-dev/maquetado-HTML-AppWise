# 🚀 Appwise Academy | Prácticas y Desafíos de HTML5

¡Bienvenidos al módulo de Fundamentos Web!

Este repositorio es tu campo de entrenamiento. Durante las clases en vivo, el profesor construirá proyectos desde cero para explicar los conceptos y la teoría (¡como nuestro clon de YouTube!). Sin embargo, **aquí es donde tú te ensucias las manos**.

En este repositorio encontrarás **10 proyectos estratégicos**. Cada uno está diseñado no solo para que domines las etiquetas HTML, sino para prepararte el terreno: estos mismos proyectos evolucionarán más adelante cuando aprendamos CSS y JavaScript.

---

## 🛠️ ¿Cómo usar este repositorio? (Flujo de Trabajo)

Para empezar a trabajar, **NO** clones este repositorio directamente. Debes crear tu propia copia privada siguiendo estos pasos:

1. Haz clic en el botón verde **"Use this template"** (Usar esta plantilla) en la parte superior derecha de esta página en GitHub.
2. Selecciona **"Create a new repository"**.
3. Ponle un nombre a tu repositorio (ej: `mi-nombre-appwise-html`) y ponlo en **Privado** o **Público** (como prefieras).
4. Una vez creado en tu cuenta, abre tu terminal y clónalo en tu computadora:
   ```bash
   git clone <URL-DE-TU-NUEVO-REPOSITORIO>
   ```
   Abre el proyecto en Visual Studio Code y ¡a programar!

## 📁 Estructura del Repositorio

Dentro de este repositorio encontrarás una carpeta por cada desafío. Debes ir resolviéndolos en orden:

```text
📦 Curso-HTML-Appwise
┣ 📂 01-Receta-Abuela/ (Textos, listas y semántica base)
┣ 📂 02-Blog-Hacker/ (Maquetado semántico avanzado)
┣ 📂 03-Registro-Usuario/ (Formularios e inputs)
┣ 📂 04-Landing-Producto/ (Multimedia y estructura comercial)
┣ 📂 05-Gestor-Tareas/ (Preparando el DOM para el futuro)
┣ 📂 06-FAQ-Acordeon/ (Interactividad nativa)
┣ 📂 07-Directorio-Devs/ (Tablas y datos estructurados)
┣ 📂 08-Carrito-Compras/ (Layout de componentes)
┣ 📂 09-Buscador-Mapas/ (Iframes y embebidos)
┗ 📂 10-Dashboard-Admin/ (El Jefe Final: estructura compleja)
```

## 📜 Reglas del Juego

- **Lee las instrucciones:** cada carpeta (ej: `01-Receta-Abuela`) tiene su propio `README.md` con las instrucciones exactas de lo que debes construir.
- **Usa la Biblia:** no intentes memorizar todo. Si olvidas un atributo, tu mejor amigo es MDN Web Docs.
- **Paciencia visual:** recuerda que ahora estamos viendo solo el "esqueleto" (HTML). Se verá feo y básico. ¡Es normal! La magia y los colores llegarán en el módulo de CSS.
- **Romper cosas está bien:** la única forma de aprender a programar es equivocándose. Prueba, borra, guarda y vuelve a intentar.
